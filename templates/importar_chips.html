Importar Chips via CSV

Instruções para Importação

  • O arquivo deve estar no formato CSV (valores separados por vírgula ou ponto-e-vírgula).
  • A primeira linha deve conter os cabeçalhos: Numero, Funcionario Nome, Centro de Custo, Funcao, Valor, Vencimento Fatura, Status.
  • Cada linha subsequente deve conter os dados de um chip.
  • O campo Numero é obrigatório - linhas sem este campo serão ignoradas.
  • Chips com números já existentes no sistema serão ignorados.
  • Centros de custo não existentes serão criados automaticamente.
  • O status deve ser um dos seguintes: Ativo, Cancelado, Em estoque.
